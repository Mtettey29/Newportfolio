Featured Project

Optimizing Network Infrastructure & Security

Led comprehensive OSI model-based analysis of existing university network infrastructure, identifying critical performance bottlenecks and security vulnerabilities. Authored primary design for hierarchical network architecture with VLAN segmentation strategy.

Problem: University network infrastructure experiencing performance bottlenecks and security vulnerabilities
Solution: Comprehensive OSI model-based analysis with hierarchical network architecture design
Technologies: Wireshark, VirtualBox, OSI Model, VLAN Segmentation, IP Subnetting (VLSM)
Results: Projected conservation of 3,400 IP addresses, identified 99.2% ARP broadcast dominance, analyzed 1.5M packets
